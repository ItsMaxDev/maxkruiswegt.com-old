Chief Bot

Chief Bot is a Discord bot written in C# using DSharpPlus. The bot is designed to have several features, including message logging, study time logging, a scoreboard, and a highly customizable pomodoro command. Chief Bot uses a MySQL database for storing and retrieving data related to study time logging and the scoreboard.

  • The message logging feature allows server admins to keep track of user activity and moderate conversations.
  • The study time logging feature is meant for users who want to track their study time and compete with others on the server using the scoreboard.
  • The pomodoro command is highly customizable, allowing users to set their own work and break times for the timer.

Database School Project

We created a C# Windows Forms program that interacts with an MSSQL database for my school project "Project Databases." The program has four parts: front-end, service layer, DAL, and Model layer. The program consists of the following elements:

  • Students (list all students)
  • Lecturers (list all lecturers)
  • Activities (create, update, delete)
  • Rooms (list all rooms)
  • Supplies (update stock and name)
  • Cash Register (select student and drink)
  • Revenue Report (choose date range, shows total sales, turnover, customers)
  • Supervisors (link to activity)
  • Participants (link to activity)
